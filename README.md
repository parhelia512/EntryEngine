# EntryEngine
### 入口引擎开发环境
1. 开发框架：.net framework 3.5，未引用其它第三方框架，理论可直接编译到.net core，standard或其它
2. 编码工具：Visual Studio 2010，可用任意更高版本

### 入口引擎可用于的开发场景
 **1. 服务端** 
    包含TCP，HTTP，WebSocket
 **2. 2D游戏客户端** 
    包含基于Xna(客户机无需安装Xna)发布的Windows PC平台(.exe)
    包含基于Unity3D可导出的所有平台
