Snippet是代码缩写补全功能，例如vs2010输入for，按两下Tab键，会输出以下代码
for (int i = 0; i < length; i++)
{

}

vue.code-snippets用于Visual Studio Code (vscode)
vue.snippet用于Visual Studio任意版本
导入编辑器后可以输入vue，按两下Tab键，输出vue模板代码

vscode导入：
点击左下角 设置图标按钮 -> 用户代码片段 -> 新建代码片段(或已有代码片段)，将vue.code-snippets代码粘贴保存即可

vs导入：
点击导航栏 工具 -> 代码段管理器 -> 导入 -> 选择vue.snippet即可
