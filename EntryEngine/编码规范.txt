若方法体中注释或字符串包含{符号，一定也要有相应的}符号，保证在生成代码时不出错
public void Method()
{
	// {	注释包含 {
	// }	一定要有 }
}

深度调用的静态类__开头，其它同静态类规范 __INPUT __GRAPHICS
